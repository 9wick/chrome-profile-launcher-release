# Chrome Profile Launcher

<img src="lp/public/icon.png" alt="Chrome Profile Launcher icon" width="160" />

## Introduction

Chrome Profile Launcher is a lightweight desktop tool to switch Chrome profiles by use case.
Pick your work, personal, or testing profile and launch Chrome instantly with the right URL.

## Features

- One-click profile switching
- Launch with specific URLs and profiles
- Lightweight, no background process
- macOS support

## Supported OS

- macOS
- ( maybe works on Windows, but not tested )

## Requirements

- Google Chrome (stable channel)

## How to Use

1. Download the latest binary from Releases.
2. Register your Chrome profiles.
3. Launch with a profile or URL.
