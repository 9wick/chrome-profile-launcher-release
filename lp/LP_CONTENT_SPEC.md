# LP Content Specification

## Core Message

**主要課題**: リンクをクリックした時にプロファイルを選択できない

**メインメッセージ**:
- 間違ったアカウントで開かない
- 複数profile運用を簡単に（仕事とプライベートの分離、複数の仕事アカウント）

**解決方法**:
デフォルトブラウザにこのアプリを設定 → アプリがprofile指定してchromeを開く

## Key Messaging Points

1. **課題の明確化**:
   - 「リンクをクリックした時にプロファイルを選択できない」が本質
   - 権限エラーやプライバシー漏洩は結果（症状）であって、直接的な訴求ポイントではない

2. **具体的なシチュエーション**:
   - メールのリンク
   - Slackやチャットのリンク
   - Notionやメモアプリのリンク

3. **ターゲットユーザー**:
   - 仕事とプライベートを分けている人
   - 複数の仕事アカウントを持っている人

4. **解決の仕組み**:
   - デフォルトブラウザに設定することでリンククリックをインターセプト
   - プロファイル選択画面を表示
   - 選択したプロファイルでChromeを起動

5. **製品の特徴**:
   - 軽量
   - 常駐なし
   - シンプル
